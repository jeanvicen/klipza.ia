# 🌟 klipza.ia - Starborne Garden

O **klipza.ia** é um assistente pessoal inteligente e minimalista, projetado para oferecer uma experiência de conversação fluida, segura e esteticamente agradável. Inspirado no cosmos e no jardim estelar, o app combina tecnologia de ponta com um design focado no usuário.

## 🚀 Principais Recursos

- **🤖 Inteligência Avançada:** Integrado com a API da OpenRouter (Llama 3.2), oferecendo respostas rápidas e precisas.
- **📱 PWA Nativo:** Instale o klipza.ia diretamente no seu celular ou PC como um aplicativo nativo.
- **🔄 Atualizações Automáticas:** O app se mantém sempre atualizado de forma silenciosa e eficiente.
- **✨ Interface Imersiva:** Design dark mode com fundo animado de estrelas, animações de escrita lenta e ícones personalizados.
- **🛡️ Segurança & Privacidade:** Controle total sobre seus dados, com opção de exclusão permanente de conta e histórico.
- **⚡ Performance Mobile:** Otimizado para rodar de forma suave em qualquer dispositivo, com foco em baixo consumo de bateria.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **PWA:** Service Workers, Web App Manifest
- **IA:** OpenRouter API (Llama 3.2 Instruct)
- **Animações:** Canvas API, CSS Transitions & Keyframes

## 📲 Como Instalar

1. Acesse o site através do seu navegador mobile.
2. Aguarde o banner de instalação aparecer ou acesse as opções do navegador e selecione **"Instalar Aplicativo"**.
3. O ícone do **klipza.ia** aparecerá na sua tela de início, pronto para uso!

## 📦 Estrutura do Projeto

- `index.html`: Núcleo da interface e lógica do app.
- `sw.js`: Service Worker para suporte offline e atualizações.
- `manifest.json`: Configurações de PWA e ícones.
- `assets/icons/`: Ícones e logos do aplicativo.
